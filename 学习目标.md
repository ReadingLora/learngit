## 2026年4月27日（周一）

今日完成：
- 安装 Git，完成第一次 commit 和 push 到 GitHub
- 配置 SSH key，连接远程仓库
- 廖雪峰 Git 教程学到 7.2 分支管理/解决冲突

今日卡点：
- git add 需要完整文件路径，直接写文件名找不到文件（文件在子文件夹时）
- git push 写了 master 报错，原因是新版 Git 默认分支名是 main 不是 master
- git diff 查不到内容，原因是已经 git add 过了，diff 只显示未暂存的变化
- 空文件夹不会出现在 git status 里，Git 只追踪文件不追踪文件夹
- cd 到错误文件夹里建了分支，需要 cd .. 回到上级目录重新操作

今日感受：
- 初步学习，还是比较有趣的。现在有AI，不会的问题也可以随时提问，令人感觉比较安心。
